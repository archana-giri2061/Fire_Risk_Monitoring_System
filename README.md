# Fire_Risk_Monitoring_System
AI-driven wildfire risk prediction system using real-time weather data, MySQL storage, and machine learning to provide early warnings and risk alerts.

## Tech Stack
- Node.js & Express
- Postgresql
- Python (scikit-learn, XGBoost)
- React (Dashboard)
- REST APIs
- TypeScript 
- Pg (node-postgres)
- Zod (validation)
- Open-Meteo API

# Project workflow
Weather API → MySQL Database → ML Model → Risk Prediction → Dashboard + Alerts
